# Weekly Review

## User-invocable
When the user types `/weekly-review`, run this skill.

## Process
1. Count completed vs added items this week
2. Identify carryovers and stale waiting items
3. Present review with velocity metrics
4. Generate 3-5 bullet manager update
5. Archive completed items older than 7 days (with confirmation)
6. Set next week priorities
7. Fetch next week's calendar for meeting prep

## Style
- No emojis, polished manager update
- Be direct about carryovers
